# Two Brother's Property Management

Website for a local lawn care and property management business in Salyersville, KY.

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Phosphor Icons

## Development

```bash
npm install
npm run dev
```

## Deployment

Push to GitHub and connect to Vercel. No environment variables needed for the MVP.
